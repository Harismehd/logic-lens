# Logic Lens Pro: Quick Reference

## Core Identity
**"An IDE that talks back."** It's a pedagogical compiler that uses errors as teaching moments to prevent "vibe coding."

## Tech Stack (The "Big Four")
1. **React**: Dynamic UI & State.
2. **Monaco**: Professional Code Editor engine.
3. **Pyright**: Industrial-strength Static Analysis.
4. **Pyodide**: In-browser Python execution (WASM).

## Critical Metrics
- **Ownership**: 85% (My logic + My Design + My Pedagogy).
- **Latency**: <300ms (Real-time feedback).
- **Difficulty**: Adaptive (XP levels + Concept Escalation).

## Key Files
- `App.jsx`: The Orchestrator.
- `ErrorEngine.js`: Syntax Analysis.
- `AdvancedPedagogy.js`: Challenges & Mutations.
- `LocalMentorService.js`: Socratic AI Mentor.
- `pyright-service.cjs`: Semantic Analysis Backend.

## 3 Pillars of Logic Lens
1. **Socratic Guidance**: Don't tell, ask.
2. **Deep Verification**: Explain your code to pass.
3. **Variation Mastery**: Fix it 3 times to know it once.

## Unique Value Prop
"Logic Lens is the only coding environment that treats a 'Syntax Error' not as a failure, but as a curriculum."
